# Angular 1.x ES6 e webpack 2

Exemplo de como configurar ambiente para desenvolvimento com Angular1.x ES6 e Webpack 2

### Prerequisites

NodeJs v6.10.1 +
npm v4.2.0 +

### Installing

npm i
npm run start
 http://localhost:9000

## Running the tests

npm run test
